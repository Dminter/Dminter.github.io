使用 freeline 加快 android 编译

发表于 2017-03-07

简介

https://yq.aliyun.com/articles/59122

安装plugin

as plugin 在线搜索freeline并安装

配置gradle

android主gradle
apply plugin: 'com.antfortune.freeline'
project主gradle
  dependencies {
    classpath 'com.antfortune.freeline:gradle:0.8.6'
    classpath 'com.mcxiaoke.gradle:packer-ng:1.0.8'
}

安装Python2.7

https://www.python.org/downloads/windows/

gradlew initFreeline

gradlew initFreeline 初始化

点击freeline plugin运行

明显比Instant Run快，且增量更新，界面即可变化，唯一缺憾是不能debug，不过界面搭建阶段巨快。
