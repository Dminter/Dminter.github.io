Terminal Help

摘录自linux终端的帮助文档。以GNOME 桌面的终端模拟器为例。

Copy and paste

复制Ctrl+Shift+C，粘贴Ctrl+Shift+V

Use tabs

打开一个新终端 Shift+Ctrl+T.

Text selection

双击选中单词，一行的空白处双击，选中整行

Open or copy web addresses

命令含有邮箱地址或者网址，右键菜单可以看到打开链接发邮件的菜单项，Ctrl+单击可以亦可

Use Find

菜单上面的搜索可以搜索命令行里面的结果
find 3.txt find 3.*

Change cursor

改变光标样式|⎕_

Terminal sizes

设置命令窗体宽高
Alt+T可以拖动窗体位置

Windows and tabs

Zoom in and zoom out

Ctrl+shift++. 放大
Ctrl+-.缩小
Ctrl+0.还原

Fullscreen session
