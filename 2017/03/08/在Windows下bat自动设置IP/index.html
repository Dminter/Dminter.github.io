在Windows下bat自动设置IP

发表于 2017-03-08

通常情况都是利用路由器DHCP功能自动获取IP地址，可是往往在公司环境，DHCP功能开启会导致不同网段相互干扰导致一片区域均不可上网的问题，因而，网管索性关闭了DHCP，转而叫同事各自设置自己的IP，Windows虽然有一套设置IP的方法，也干不翻做IT的午我们，但是，也不乏有不会用的，对此利用Windows命令行参考网上教程写了一个bat解决此困扰。

IP末尾取值

4个字节，0~255，暂不讨论此IP已被占用，需要一个随机末尾数。

随机数

set /a n=%random%%%255+1
echo %n%
pause

设置DNS

netsh interface ip set dns name="本地连接" source=static addr="223.5.5.5"  register=PRIMARY
netsh interface ip add dns name="本地连接" addr="223.6.6.6" index=2
pause

IPsetting.bat

set /a n=%random%%%255+1
netsh interface ip set address name="本地连接" source="static" addr="192.168.1.%n%" mask="255.255.255.0" gateway="192.168.1.254" gwmetric="1"
netsh interface ip set dns name="本地连接" source=static addr="223.5.5.5"  register=PRIMARY
netsh interface ip add dns name="本地连接" addr="223.6.6.6" index=2
pause

注意

若网络连接中，含有多个网络适配器需要指定一个上网的为“本地连接”
